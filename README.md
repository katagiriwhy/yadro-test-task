# Приветствую команду YADRO!

Моя реализация тестового задания. Ниже представлены инструкции по сборке, запуску и тестированию проекта.

---

## Build

> [!NOTE]
> The project uses ```C++17``` standart.

### Сборка и запуск
```bash
make build
```

### Сборка и запуск тестов
```bash
make tests
```

### Удаление директорий build
```bash
make clean
```
